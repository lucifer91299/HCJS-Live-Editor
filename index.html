INTRODUCTION


HCJS-Live Editor will be using HTML, CSS, And JavaScript with the help of Ace Editor.

Description of the System:

This project is a live webpage editor in which the user can write input of the code and see its output simultaneously. HCJS-Live Editor will be using HTML, CSS, and JavaScript with the help of Ace Editor. It is simple to understand and user friendly.
The most discouraging situation for a programmer is when he is typing a code and has to check its output each and every time by compiling and running it. Every time he makes a change in his code he has to again run it and look at the output. This can be very tedious and time-consuming. Even more when the client writes an entire code only to find it not working.


HCJS-Live Editor completely relieves the programmer from this task. It provides two areas to the client, one for coding and the other for displaying its output. While writing, client sees the output generated by his code at the same time. He does not need to wait for the program to run or compile. There is no button that he needs to press. The client will only write his code and the editor will do the rest.
The user can add formatted text, equations, images, and hyperlinks to enhance the narrative. The HCJS-Live Editor also shows the error if there is any, while writing code. Various form methods like pop-up text, labels, etc. are used to further guide the user.
The Editor will detect each line of code and will compile it at the same time. Each line of code is run and the output of the same is displayed in the output area. As for html statements, the outputs will not be displayed if there were any logical errors in the code. But the editor detects and notifies the user if there are any syntactical errors.

This work on only HTML, CSS, And JAVASCRIPT

FEATURES: -

  1. Live Editor
  2. Syntax highlighting
  3. Automatic indent and outdent
  4. An optional command line
  5. Handles huge documents (four million lines seems to be the limit!)
  6. Live syntax checker (currently JavaScript/Coffee Script/CSS/XQuery)
  7. Cut, copy, and paste functionality
  8. Toggle between soft tabs and real tabs
